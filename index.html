Welcome to CPE Online

      
    

    

      
A C / C++ / Python IDE that runs on your phone — no internet needed after the first visit.

      
    
        
  1. Edit your code in the editor. Tap the symbol bar above the keyboard for { } ; ( ) etc.
    2. 
        
  2. Need input? Type it into the Input row at the bottom of the Output panel before pressing Run (for C / C++).
    3. 
        
  3. Tap ▶ Run to compile and run. Output appears in the bottom panel.
    4. 
        
  4. Save downloads your file as .c, .cpp, .py, or .java. Open brings them back.
    5. 
      

      
💡 Tap the ☰ menu for examples, snippets, themes, and the full help.
